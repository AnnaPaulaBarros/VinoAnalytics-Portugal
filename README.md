# VinoAnalytics Portugal

Este projeto realiza a estruturação e análise visual da produção vinícola de Portugal, integrando SQL Server, Python e Streamlit.

#Funcionalidades
- Pipeline de Dados Híbrido: O sistema consome dados de um banco SQL Server local ou de um arquivo CSV.
- Geovisualização: Mapa interativo utilizando Plotly com as cores da bandeira de Portugal.
- Filtros: Análise por região e faixa de produção em hectolitros (hl).

#Tecnologias Utilizadas
- Banco de Dados: SQL Server (T-SQL para estruturação idempotente).
- Linguagem: Python.
- Bibliotecas: Pandas, PyODBC, Plotly e Streamlit.
